# PerioVoice AITM

## Overview

PerioVoice AITM is an AI-driven conversational system designed to assess periodontal (gum) disease symptoms and recommend the urgency of visiting a dentist. The system combines voice interaction, AI-based symptom analysis, image analysis, and risk prediction to help users identify potential gum problems at an early stage.

## Features

* User Registration and Secure Login
* AI-Based Symptom Assessment Chatbot
* Voice and Text Input Support
* Gum Image Analysis
* Periodontal Risk Prediction
* Personalized Recommendations
* Report Generation and Download
* User Profile Management

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Database: MySQL
* AI/ML: Python, Scikit-learn
* NLP & Speech Recognition
* GitHub
* Docker
* Kubernetes
* Jenkins

## Modules

1. User Authentication & Profile Management
2. Symptom Assessment Chatbot
3. Image Analysis & Risk Prediction
4. Report Generation & Recommendations

## Objective

The objective of this project is to help users identify periodontal disease at an early stage by using AI to analyze symptoms and provide urgency recommendations for dental consultation.
